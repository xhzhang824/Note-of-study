在程序分配空间的时候，一般会栈区和堆区的之分，但是究竟什么是堆区什么是栈区呢，对于栈区，就是结构形式类似于栈的一块存储区域，在这块存储区域中申请空间来存放变量的时候一般是按照连续申请和存放的，也就是说在这样的形式下我们如果申请的空间大于栈本身的空间的话那就会overflow了；而所谓的堆区就是我们的操作系统为我们生成的一些存储空间，这里说一些还是很有道理的，就是因为这些存储空间并不是连续的，它是我们的操作系统为我们收集的一些空闲的空间，它们是以链表的形式关联起来的，（特别注意的一个问题 就是我们这里所说的堆并不是数据结构中的堆）
在c语言中，
如下：
int	a=0;
int A[1000];
functionA()
{
	int b;
}
//以上的a，A，b这些变量都是放在栈中的

如下：
int *p=(int)malloc(int*40);
这样的变量指向的存储空间就是堆，
****但是***很重要的一点就是这里的p却是分配在栈中的！！！
